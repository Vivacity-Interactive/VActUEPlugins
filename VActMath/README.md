# VActMath
ALl math functions for pointer type arrays, to use with `reinterpret_cast<float*>(&MyFloatPropertyStruct)` or just (float*)&MyFloatPropertyStruct. Functions like `Dot`, `Lerp`, `Clamp` and more. [VActCamera](VActCamera/README.md)
